- Proposta de projeto escolhida: Sistema web para divulgação e busca de veículos à venda (AutoCar).
- Breve descrição sobre seu projeto: O projeto AutoCar consiste no desenvolvimento de uma plataforma web que permite aos usuários visualizar, buscar e explorar veículos disponíveis para venda. A aplicação apresenta carros em destaque e anúncios recentes, organizados em um layout moderno e responsivo.

O sistema conta com um menu de navegação, campo de busca e seções específicas para exibição dos veículos, contendo informações como modelo, ano e preço. O objetivo é facilitar a experiência do usuário na procura por carros, tornando o processo mais rápido, intuitivo e acessível em diferentes dispositivos.
